## Setup

### Requirements
* [NodeJS](https://nodejs.org/en/) (v9.4.0 +)
* [Gulp Cli](https://gulpjs.com/) (v2.0.1 +)
* Some IDE

### Get started
* Run `npm install`
* Setup `.env` File with your bot token:
    ```
    token = YOUR_TOKEN
    ```
* Setup eslint in your IDE
* Run `gulp` for build process
